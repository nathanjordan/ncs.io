What is NCS?

NCS (short for Neo-Cortical Simulator) is an open-source neural simulation tool. Like other neural simulators such as NEURON, GENESIS, PyNEST, and BRIAN, it allows neuroscientists to to accurately perform experiments on biologically-realistic neural networks without going in vivo or in vitro. The simulator lets users build, test, and visualize models using a comprehensive set of web-based tools.

What makes it different?

NCS has many features that distinguish it from other neural simulators. Unlike simulators such as NEURON and GENESIS, NCS focuses on simulating large-scale neural networks: on the order of millions of neurons, while still maintaining biological accuracy. NCS also includes a wide-array of tools to help in the development of these large models, such as an easy-to-use web interface, an integrated model database that lets you share models with other researchers, and the ability to deploy a simulation on the Amazon Cloud if you don't have your own cluster.

What can it do?

NCS can run on one computer, or a cluster of over a hundred. It runs on both the CPU and NVIDIA GPU's (using CUDA) to generate maximum computational power. Using a sufficient number of machines and NVIDIA GPU's, the simulator can run a million-cell model in real time. The simulator also offers different neuron models within the same simulation, including Leaky-Integrate and Fire, Izhikivich, and Hodgkin-Huxley to provide neuroscientists the ability to select either detail or performance in different parts of the simulation.
